# 🛒 ScannyCart – Smart Barcode-Based Self-Checkout System

## 📌 Description
A smart barcode-based system that allows users to scan products and fetch real-time product details using external APIs.

## 🚀 Features
- Barcode scanning
- Real-time product info
- Add to cart functionality
- JWT authentication
- Role-based access

## 🛠️ Tech Stack
- Java
- Spring Boot
- PostgreSQL
- JavaScript

## ▶️ How to Run
1. Clone the repo
2. Configure database
3. Run the Spring Boot application
